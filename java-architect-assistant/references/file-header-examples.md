# File Header Examples

Use these examples only as a fallback when the repository has no obvious existing header template.

## Principle

If nearby files already contain a header comment, copy that style and only replace the business-specific parts.

Only use the following examples when no stable local pattern can be found.

## Java Example

```java
/**
 * Author: oygj
 * Description: [brief description]
 * Date: 2026-03-23
 */
```

## SQL Example

```sql
-- Author: oygj
-- Description: [brief description]
-- Date: 2026-03-23
```

## XML Example

```xml
<!--
  Author: oygj
  Description: [brief description]
  Date: 2026-03-23
-->
```

## Important Constraint

These are fallback examples, not mandatory universal headers. Local project style always wins.
